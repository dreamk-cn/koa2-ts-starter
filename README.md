# Koa2 TypeScript Starter

这是一个基于 Koa2 + TypeScript + pnpm 的后端起始项目模板。

## ✨ 特性

- **TypeScript**: 全量类型安全 (`Node16` 模式)
- **别名支持**: 使用 `@/` 引用 `src/` 目录 (开发与生产环境均支持)
- **代码规范**: ESLint + Prettier 自动化配置
- **工程化**: 统一的响应处理、全局异常捕获、环境变量管理
- **包管理**: 使用 pnpm 极速安装

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/dreamk-cn/koa2-ts-starter.git my-project
cd my-project
